# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## 0.3.0

### Added

- [#8](https://github.com/mylu0x/texto/pull/8): Add `uuid` command to generate UUIDs (includes v4 and v7)
- [#10](https://github.com/mylu0x/texto/pull/10): Add `-o` / `--output` and `--force` global arguments
  - `-o` / `--output`: Output the result to a file
  - `--force`: Force overwrite if the output file already exists

## 0.2.0

### Added

- [#4](https://github.com/mylu0x/texto/pull/4): Add No, Ru, Pl and It languages to the `text` command
- [#5](https://github.com/mylu0x/texto/pull/5): Add `lorem` command to generate Lorem Ipsum
- [#6](https://github.com/mylu0x/texto/pull/6): Add `--format` / `-f` argument to the `text` command (plain, html and json)
